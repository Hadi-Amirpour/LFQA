# LFQA
Light field quality assessment code

This code is used to measure full-reference objective quality metrics between sub-aperture images. 


# inputs
  - Directory of uncompressed and compressed sub-aperture images.


# outputs
  - PSNR
  - SSIM
  - MS-SSIM
